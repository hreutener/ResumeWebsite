Intro

Hello and thank you for visiting my portfolio website! I am currently a student at Shawnee State University and will be graduating Spring of 2023. I will be obtaining a Bachelor of Science Degree in Digital Simulation/Game Design & Computer Engineering with a Minor in Sound Design.

Growing up I spent a lot of my free time playing call of duty, minecraft, and super smash brothers. Playing these games made me curious about how they were developed and created. I then began researching coding and became hooked. I enjoyed the logic used in coding and "making things work" both in game development and other softwares. Studying game programming in college not only strengthened my programming with games but taught me many techniques used in software engineering. With my major in computer engineering technologies I also learned about the hardware side of technologies and how software and hardware must work simultaneously. I have a well rounded knowledge of how components work together to make a piece of technology. With this experience of constructing and coding I have been able to create working systems. During my freshman year I took a course in sound design. This course resonated with me, leading to the addition of a minor in sound design as well. Through my college career my passion for software engineering grew and I knew this was the vocation I wanted to pursue.

Past Work

Below you can find projects I have worked on using various softwares and techniques.



B.L.A.S.T.

B.L.A.S.T is a multiplier 1v1 first person shooter game which was the senior capstone project (2022) for the Digital Simulation/Game Design portion of my degree. It is a vertical slice that was made by a group of 20 students composed of both artists and programmers in Unreal Engine 4 using C++. My roles for this project were audio engineer, sound designer, and a member of the QA team. To create the sounds and effects I used Logic Pro X and Unreal Engine’s built in audio engine.

RecipeBook Android app

This android app, created with Android Studio using Kotlin, is a recipe app that allows the user to enter recipes, including ingredients and steps to increase both the ease of shopping and cooking. The app stores the recipes in a database locally using android’s room database scheme. The user is also able to have their shopping list contained on the app with the option to add the ingredients from their recipes to their shopping list as they need. I hope to continue to build this app by adding additional features such as, improved visual design and create a database to input ingredients from your kitchen to select relevant recipe ideas.

Cryptarithm Solver

This Python program is designed to find all possible solutions for a given Cryptarithm. A Cryptarithm is a puzzle where words can be made cryptic by replacing a given statement’s letters and changing each letter to a number that is unique. An example of this would be: SEND + MORE = MONEY where the only solution would be 9567 + 1085 = 10652.

Cooking Simulator Overdub Video

This project created in Logic Pro X overlaid audio for the game trailer of Cooking Simulator VR. This audio was was recorded using a shotgun microphone in combination with a condensor microphone. I also created the background music in the video using samples from Logic Pro X.

Current Work

Here are a few projects I am actively working on.



Boxen

Boxen is a mock boxing game made in Unreal Engine 4 where you play as a cardboard box trying to knock the opposing box out of the ring. You control a box with WASD and mouse using the moves attack and block. Each time you push the enemy box, or the enemy box pushes you, the pushed box loses some stability and will get launched a bit until eventually one is knocked out of the ring. Future plans for this game include improving the AI for the enemy box, adding the option for multiplayer, and adding textures.

Sythesizer and Sequencer

My senior project for my Computer Engineering Technologies major is a synthesizer with a sequencer. I am creating this by using C++ with an arduino mega 2560 board, capacitive touch buttons, switch buttons, and speakers. The arduino sketch uses the tone library to play the tones. The device will have a single octave of notes that can be increased or decreased as desired with a four channel looping sequencer that can play along with it. Each note in the looping sequencer can be set to play any tone desired from the octave of notes along with the speed of the sequencer able to be increased or decreased. I also hope to 3-D model a case for this device to be housed.
